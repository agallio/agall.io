Hai, saya Aga 👋🏻 . Saya seorang web engineer, musisi dan juga kreator aplikasi web [FreedomLife](https://freedomlife.id). Saat ini saya sedang bekerja sebagai Principal Frontend Engineer di [PrivyID](https://privy.id).

Saya mulai menyukai dan belajar web engineering sejak berusia 12 tahun. Saya terkagum dengan bagaimana cara internet bekerja pada saat itu. Aplikasi-aplikasi sosial media besar berbasis internet seperti [Facebook](https://facebook.com) & [Twitter](https://twitter.com) dengan segala fitur dan kontennya membuat saya tertarik untuk terjun ke dalamnya.

Tak lama setelah kejadian saya mengenal internet khususnya aplikasi sosial media itu, ayah saya menuntun saya untuk berjelajah lebih jauh lagi dengan memperkenalkan saya ke dunia web engineering. Ayah saya yang memang sedari dulu adalah _computer geek_ memperkenalkan banyak hal tentang bagaimana sebuah halaman website bisa tersedia di internet. Sampai di satu titik ayah saya menjelaskan tentang bagaimana **cara membuat website** saya langsung tertarik dan ingin langsung mencobanya.

Singkat cerita, saya membuat sebuah halaman website statis sederhana yang berisi sintaks yang sangat populer pada zamannya yaitu `<marquee>` 😂 . Sudah berasa paling keren dari teman-teman seumur saya 😅 . Tentu saja saya di tuntun oleh ayah saya untuk membuat website tersebut. Setelah kejadian tersebut, saya akhirnya lebih sering untuk berjelajah di internet bukan hanya untuk membuka sosial media saja, namun juga mencari cara-cara untuk membangun sebuah aplikasi web.

## Pendidikan

Perjalanan saya berlanjut, saya memutuskan untuk masuk ke Sekolah Menengah Kejuruan ([SMKN 2 Depok Sleman](https://smkn2depoksleman.sch.id/wp/)) dengan jurusan Teknik Komputer Jaringan untuk melanjutkan ketertarikan saya terhadap dunia internet juga IT. Saya ada di sekolah selama 4 tahun. 3 tahun sekolah, 1 tahun bekerja secara profesional. Pada saat kelas 2 & 3, saya diberi kesempatan untuk mengikuti Lomba Kompetensi Siswa SMK atau biasa disebut dengan [LKS SMK](https://id.wikipedia.org/wiki/Lomba_Kompetensi_Siswa). Saya ikut serta dalam bidang lomba Web Design. Saya banyak belajar tentang bagaimana web engineering berkembang dari waktu ke waktu, juga library-library baru yang mulai banyak diadopsi oleh para web engineer seperti [jQuery](https://jquery.com/), [React](https://reactjs.org/), dan [Vue](https://vuejs.org/).

Saat mempersiapkan diri untuk ikut lomba, kami diberikan kisi-kisi oleh panitia [LKS SMK](https://id.wikipedia.org/wiki/Lomba_Kompetensi_Siswa) yang berisi kami harus membuat sebuah aplikasi web yang sudah ditentukan oleh mereka dan ternyata banyak persyaratan yang harus diikuti. Salah satunya adalah kami harus membuat aplikasi web tersebut menggunakan framework-framework full-stack yang saat itu sedang populer seperti [Laravel](https://laravel.com/) (PHP + Blade), [Yii](https://www.yiiframework.com/) (PHP + Twig / Smarty), juga [MERN stack](https://www.mongodb.com/mern-stack) (MongoDB, React.js, Express.js, Node.js). Karena saya sudah terbiasa dengan JavaScript, akhirnya saya memilih [MERN stack](https://www.mongodb.com/mern-stack). Singkat cerita, puji Tuhan setelah jerih payah selama kurang lebih 6 bulan dilatih dan dikarantina, saya mendapatkan juara kedua.

Saya sangat bersyukur telah diberi kesempatan untuk mengikuti lomba tersebut. Saya belajar banyak sekali tentang pemrograman, web engineering juga mencicipi sedikit bagaimana hidup di dalam dunia kerja.

## Pekerjaan

Setelah saya selesai masa 3 tahun belajar di sekolah, saya masuk ke masa 1 tahun bekerja secara profesional. Saat itu saya memilih untuk mencari perusahaan sendiri dan tidak dibantu sekolah. Perjalanan saya menjadi seorang profesional dimulai dari :

1. **Frontend Engineer**, **Docotel Group** (Jun 2018 - Mei 2019)

   Saya membantu tim untuk membuat sebuah produk dalam domain _human resources_. Saya dipercaya untuk ikut serta dalam seluruh lifecycle produk tersebut, dari awal diskusi dengan tim hingga produk jadi. Saya membuat kerangka awal codebase frontend nya, memutuskan bersama tim design language yang akan digunakan, dan menuangkannya ke dalam kode. Saya juga mengintegrasikan API services, dengan codebase frontend yang saya buat. Saya belajar bagaimana memulai sebuah produk dari awal dengan memperhatikan best practices dan efektivitas dalam bekerja.

2. **Frontend Engineer**, **Skyshi Digital Indonesia** (Jun 2019 - Des 2019)

   Saya membantu sebuah startup multi-nasional bernama [GetCraft](https://getcraft.com). Saya bekerjasama dengan lebih dari 20 orang yang berada di dalam maupun di luar negeri. Pengalaman tersebut membuat saya belajar banyak tentang bekerja dalam tim dengan berbahasa Inggris. Banyak hal & pandangan baru yang saya dapat. Salah satunya adalah cara bekerja secara remote. Saya berada di dalam _Chat Squad_, yang pekerjaan sehari-harinya adalah membangun fitur chat di dalam aplikasi web [GetCraft](https://getcraft.com) tersebut.

3. **Frontend Engineer**, **PrivyID** (Des 2019 - Jun 2020)

   Saya dipercayakan untuk membuat banyak aplikasi web yang mengintegrasikan SDK PrivyID (identitas & tanda tangan digital) dengan kebutuhan klien. Beberapa contohnya adalah aplikasi web pengajuan [Mandiri Kartu Kredit](https://apply.mandirikartukredit.com/) dan [BNI Kartu Kredit](https://applykartukredit.bni.co.id/). Saya banyak belajar tentang proses di dalam sistem perbankan dan multifinance. Pandemi COVID-19 membuat banyak bank & multifinance datang untuk membuat aplikasi web untuk pengajuan pengunduran pinjaman atau bahkan pengajuan pinjaman bagi UMKM yang bisnisnya menurun akibat pandemi.

4. **Principal Frontend Engineer**, **PrivyID** (Jul 2020 - Sekarang)

   Saya dipercayakan untuk memimpin satu tim Frontend Engineer. Saya belajar bagaimana menjadi seorang mentor dan manager. Membantu dan melatih teman-teman saya untuk lebih mengerti tentang web engineering khususnya di bagian frontend. Saya merencanakan, melakukan riset, dan membangun arsitektur frontend untuk mempercepat dan mempermudah pekerjaan sehari-hari. Saya membantu interaksi VP dan Engineering Manager dengan klien dari sisi teknis, dan juga membantu interaksi mereka dengan Junior-Senior Frontend Engineer agar tercipta lingkungan yang jelas dan terbuka. Saya juga membantu tim HR dalam perekrutan engineer baru.
